  • Purpose of the application

    Web application for extracting product reviews from the Ceneo.pl website based on their individual codes and for displaying the downloaded opinions and basic statistics about them.

  • Description of the application's operation
    • Analysis of the structure of the webpage.
    • Send http request to access page with opinions.
    • Check the status code of http response.
    • Parse the html code of first page with opinions
    • Extract required data from parsed code
    • If there are more pages, move to the next page and repeat steps 2-6 for each one of them save extracted data
  • Libraries/Packages used:
    • Flask
    • Requests & BeautifulSoup4
    • Pandas & NumPy
    • Matplotlib
    • Deep_translator
    • XlsxWriter
    • Flask_WTF
    • Jinja
    • io, oi & json
  • Applied approaches

    The application's functionality is exposed through the following Flask endpoints (routes):

    • /: Renders the application's main homepage.
    • /extract (GET): Displays the form for entering a Ceneo product ID to begin extraction.
    • /extract (POST): Processes the submitted product ID, initiates the scraping, analysis, chart generation, and data saving, then redirects to the product's detail page.
    • /products: Lists all products for which data has been previously extracted.
    • /product/: Displays detailed statistics and basic information for a specific product. Includes links to export data and view charts.
    • /about: Provides information about the project author.
    • /product//tocsv: Allows downloading of a product's opinions in CSV format.
    • /product//tojson: Allows downloading of a product's opinions in JSON format.
    • /product//toexcel: Allows downloading of a product's opinions in Excel (.xlsx) format.
    • /charts/pie/: Serves the dynamically generated pie chart image for a given product ID.
    • /charts/bar/: Serves the dynamically generated bar chart image for a given product ID.
    • /charts//view: Renders a dedicated page to display both the pie and bar charts for a specific product.
